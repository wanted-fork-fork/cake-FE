# react-mobile-web-boilerplate

모바일 웹을 만들기 위한 보일러 플레이트 레포지토리입니다.

## 주요 기술

- React.js: 웹 UI 라이브러리
- Next.js: 프로젝트 생성, SSR, Routing 등
- styled-component: styling
- MobX: 상태 관리
- TypeScript: 정적 타입 사용 및 코드 에러 검출
- ESLint & Prettier: 코드 컨벤션 검사
- Axios: HTTP 비동기 통신

## 프로젝트 실행

```
yarn install 
yarn dev
```
이후 `localhost:3000` 확인
